# Big Bang to Earth 3D Simulation

An interactive 3D simulation of the universe's evolution using Three.js and TypeScript.

## 🌌 Features
- **Big Bang:** The initial expansion of the singularity.
- **Quark-Gluon Plasma:** The hot, chaotic soup of fundamental particles.
- **Galaxy Formation:** The birth of spiral galaxies from cosmic dust.
- **Solar System:** A young Sun and its protoplanetary disk.
- **Earth:** A cooling planet in its cosmic context.

## 🚀 Getting Started

### Installation
```bash
npm install
```

### Run Development Server
```bash
npm run dev
```

### Controls
- **Left Click:** Rotate
- **Scroll:** Zoom
- **Right Click:** Pan
- **UI Buttons:** Switch between cosmic epochs
